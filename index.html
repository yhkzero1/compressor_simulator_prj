컴프레서 시뮬레이터 매뉴얼

컴프레서란?

Threshold를 초과하는 신호를 Ratio 비율로 압축해 다이나믹 레인지를 좁히는 장치입니다. 압축으로 낮아진 전체 레벨은 Makeup Gain으로 보상합니다.

파라미터

Threshold
−80 ~ 0 dB
압축이 시작되는 레벨. 낮출수록 압축 구간이 넓어집니다.
Ratio
1:1 ~ ∞:1
초과분 압축비. 4:1이면 초과 4 dB가 1 dB로 줄어듦. ∞:1(100:1)은 하드 리미터.
Knee
0 ~ 30 dB
전이 구간 폭. 0 = 하드(즉시 전이), 클수록 소프트(점진적 전이).
Attack
20 μs ~ 300 ms
압축 개시 속도. 짧으면 트랜지언트를 즉각 압축, 길면 어택 구간이 통과합니다.
Release
1 ms ~ 5.0 s
압축 해제 속도. 짧으면 펌핑(pumping) 아티팩트, 길면 다음 히트에 GR이 잔존합니다.
Hold
0 ~ 500 ms
릴리즈 시작 전 GR을 유지하는 시간. 빠른 연타의 채터링 방지에 유효합니다.
Source Gain
−24 ~ +24 dB
입력 전 레벨 조절. 올릴수록 Threshold를 더 자주·크게 초과해 압축량이 증가합니다.
Makeup Gain
0 ~ +40 dB
압축 후 레벨 보상. AUTO는 최대 GR만큼 자동 보정하여 원본과 유사한 레벨을 유지합니다.

컴프레서 모델

LA-2A
Optical · Tube (1962) — 어택·릴리즈가 신호 특성에 따라 자동 반응. 파라미터 조작이 단순하며 부드럽고 투명한 압축이 특징.
1176
FET (1967) — 20~800 μs의 극단적으로 빠른 어택. 트랜지언트 정밀 제어에 강점. Ratio는 4/8/12/20:1 고정값으로 선택.
SSL-G
VCA Bus (1980s) — 믹스를 하나로 묶는 "Glue" 효과. Attack 0.1~30 ms, Ratio 2/4/10:1 단계 선택. 드럼·믹스 버스에 가장 많이 사용.

사용 팁

  • Attack으로 트랜지언트 조절 — 짧게 설정하면 피크가 압축되고, 길게 설정하면 어택 구간이 통과해 펀치감이 살아납니다.
  • Release와 히트 간격 — Release가 히트 간격보다 길면 다음 히트 시작 시 GR이 남아 있습니다. Hit Spacing을 늘려 가며 차이를 관찰해 보세요.
  • AUTO Makeup 활용 — AUTO를 켜면 레벨 차이 없이 음색 변화만 비교할 수 있습니다.
  • 노브 조작 — 더블클릭으로 기본값 초기화, Shift+드래그로 미세 조절이 가능합니다.
