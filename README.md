# 교육용 컴프레서 시뮬레이터

이 프로젝트는 웹 브라우저에서 동작하는 교육용 오디오 컴프레서 시뮬레이터입니다. 다양한 드럼/베이스 샘플에 압축기 설정을 적용해보고, 파형 및 이득 감소 그래프를 시각적으로 확인할 수 있습니다.

## 주요 기능

- 6가지 샘플 소스: `kick`, `snare`, `hihat`, `tom`, `clap`, `bass`
- 컴프레서 파라미터 제어:
  - Threshold
  - Ratio
  - Knee
  - Attack
  - Release
  - Hold
  - Makeup Gain
- 입력 게인(Source Gain) 및 반복 재생(Repeat Count)
- 원본 / 컴프레스트 / 컴프레스트 + 메이크업 출력 비교 모드
- 오토 메이크업 토글
- 파형, 이득 감소(GR), 컴프레서 전송 곡선(Transfer Curve) 시각화
- 브라우저 오디오 재생 기능

## 포함된 파일

- `교육용 컴프레서.html` - 앱 HTML과 CSS 스타일
- `app.js` - 컴프레서 시뮬레이션, 렌더링, 오디오 재생 로직
- `knobs.js` - 회전식 노브 UI 컨트롤러
- `theme.js` - 테마 색상 팔레트 및 액센트 색상 선택

## 실행 방법

1. 이 저장소를 로컬에 복사합니다.
2. `교육용 컴프레서.html` 파일을 최신 웹 브라우저에서 엽니다.
3. 브라우저에서 인터페이스를 사용해 파라미터를 조절하고 `Play` 버튼으로 결과를 확인합니다.

> 참고: 일부 브라우저는 오디오 재생을 위해 사용자 상호작용이 필요할 수 있습니다. `Play` 버튼을 클릭하면 오디오가 재생됩니다.

## 사용 가이드

1. 샘플 버튼을 눌러 재생할 소스 선택
2. `Repeat` 버튼으로 히트 반복 횟수 변경
3. `Hit Spacing` 슬라이더로 히트 사이 간격 조정
4. 컴프레서 파라미터를 조절하여 압축 효과 확인
5. `Listen` 모드로 원본, 컴프레스, 컴프레스+메이크업 비교
6. `Auto Makeup` 토글을 켜면 최대 이득 감소에 맞춰 자동으로 메이크업 게인이 적용됩니다.

## 개발자 노트

- `app.js`에서 샘플 생성 및 컴프레서 알고리즘이 구현되어 있습니다.
- `knobs.js`는 마우스 드래그, 휠, 키보드 제어를 지원하는 커스텀 노브 컴포넌트를 제공합니다.
- `theme.js`는 액센트 컬러 팔레트를 렌더링하고 CSS 변수 및 캔버스 색상을 동적으로 갱신합니다.
